## 決定
タイマー表示をデジタル mm:ss から、白黒・ローマ数字・細枠の**アナログ時計（SVG）**に刷新する。残りは1本の針で表し満了で一周（経過割合 = `useTimer.progress`）。視覚的な数字は出さず、残り時間は `role="timer"` の `aria-label` でのみ公開する。

## 背景 / 根拠
- ユーザー要望（参照画像）: 白黒・余計な情報を載せない・文字少なく・枠も少なく。
- ロジックと表示が分離済み（#1）のため、刷新は `src/ui/` に閉じる。`useTimer` に `totalMs`/`progress` を追加して描画に必要な経過割合だけ供給。
- アクセシビリティと自動テストのため、数字は視覚非表示でも `aria-label` で残り時間を保持する（純粋に針だけにすると残り秒が読めない問題への対処）。
- 設定パネルは既定で折りたたみ（「設定」トグル）し、主画面を時計＋操作だけに保つ（“余計な情報を載せない”）。

## 含意 / 不採用
- プログレス円弧案・目盛り付き案は不採用（よりミニマルな「1針一周＋ローマ数字」を採用）。
- 完了ポモドーロ数の常時表示は廃止（最小表示を優先）。必要になれば非テキストのドット表示等で再検討。
- 配色は白黒固定（テーマ変数は monochrome 化）。

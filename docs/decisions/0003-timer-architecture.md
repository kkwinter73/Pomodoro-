## 決定
残り時間は `endAt`（終了予定の絶対時刻）と現在時刻の差で**毎フレーム算出**する。`setInterval` で秒を1ずつ減算する方式は採らない。

## 背景 / 根拠
- `setInterval(…, 1000)` の累積はタブ非アクティブ時のスロットリングやイベントループ遅延でドリフトする。ポモドーロは数十分を正確に測る必要があり、累積誤差は致命的。
- タイムスタンプ基準なら、タブ復帰・スリープ復帰後も `now - endAt` を引き直すだけで正しい残り時間に復元できる。
- 一時停止は「残り時間(ms)を保持」、再開時に `endAt = now + remaining` で再計算する。

## 含意
- `setInterval` は「表示更新トリガ」としてのみ使い、時間計算には使わない（誤差の出ない `Date.now()` を真実源にする）。
- 永続化は `endAt` か、停止中なら `remainingMs` を localStorage に保存。リロードで状態復元できる。
